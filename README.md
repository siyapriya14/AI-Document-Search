# AI Document Search System

This project is an AI-based document search system developed using Python and Streamlit.
It allows users to upload documents and search relevant information easily.

## Features
- Upload text documents
- Search information from documents
- Simple and user-friendly interface
- Fast results

## Technologies Used
- Python
- Streamlit
- VS Code

## How to Run the Project
1. Install Python
2. Install Streamlit
3. Run the command:
   streamlit run app.py
